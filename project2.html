Janus

The Project: This is a project we built for a hackathon. Everyone who was on this team is very passionate about music so it was pretty natural for us to make an app that defined our interest. We really wanted to make something that was not only unique but something that could be easily demoed. Hence project Janus was born. Janus uses facial recognition technology to determine a user's mood and then recommends music based off of that data.

My role: My role in this project was to work with the Microsoft Oxford Emotion API and the webcam. I started by breaking down exactly what I needed to accomplish. First, I had to take pictures using the web cam; Second, send that data to the API to get a reading; Finally, use the returned data in a way that was meaningful for our project. Being familiar with the WebcamJS library for NodeJS I immediately went to work and was taking pictures with the webcam in no time. My next task proved to be more difficult.

While I had no problem taking pictures with the web cam, the API would not accept the raw image data that was being saved. After some troubleshooting I concluded that the API needed to point to an image saved in jpeg format. The solution, I saved the image to the local file system and used a PHP script to upload that image to the server. Once the file was on the server I was able to send the URL to the Microsoft API and get a valid reading. Now that I was getting some data back from the API I needed to make it usable.

Microsoft's Emotion API returns the data in JSON format with 8 different values that measure different emotions. Since our project was recommending music based off a specific mood I needed to quantify that data and return only one emotion rather than 8. I designed an algorithm that would read the data and return a single emotion based off the prominent values.

Result: We were very happy with the results of this project. On demo day we brought up one of the judges to try out the app live before the audience.

Tech used: PHP, AngularJS, NodeJS, Microsoft Oxford Emotion API, Spotify API
